
`input_line`は`Result<String, Error>`

## `match`

```rust
  match input_line {
      Ok(line) => {
          let words: Vec<_> = line.split(" ").collect();
          println!("Line: {words:?}")
      }
      Err(_) => {}
  }
```

## `if`

```rust
fn main() {
    for input_line in std::io::stdin().lines() {
        if let Ok(line) = input_line {
            let words: Vec<_> = line.split(" ").collect();
            println!("Line: {words:?}")
        }
    }
}
```

err時に何もしないならmatchよりも単純にかけるが直感的ではなさげ。慣れてないだけかな...
